## KuCoin API Details

**Base URL:** `https://api.kucoin.com`

**Klines (OHLCV) Endpoint:**
```
GET /api/v1/market/candles?type=1hour&symbol=BNB-USDT

Response:
[
  [
    "1672531200",   // timestamp (seconds)
    "245.5",        // open
    "248.2",        // close  
    "249.1",        // high
    "244.8",        // low
    "12345.67",     // volume
    "3045678.90"    // turnover
  ],
  ...
]
```

**Rate Limits:** 
- Public endpoints: **No API key required**
- 100 requests per 10 seconds (very generous)
